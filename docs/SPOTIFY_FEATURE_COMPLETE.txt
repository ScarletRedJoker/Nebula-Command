   - Paste the URL
   - Set size: 600x200 (or customize)

TECHNICAL DETAILS:
==================

Architecture:
- Replit Spotify Connector handles OAuth
- Backend fetches current track every 5 seconds
- OBS browser source polls public API endpoint
- React component renders overlay with Tailwind CSS

Security:
- OAuth managed by Replit (no credentials in code)
- Public endpoint for OBS (read-only, no auth needed)
- All other endpoints require authentication

Performance:
- ~12 API calls/minute (one every 5 seconds)
- Minimal bandwidth (~1KB per request)
- Negligible CPU/memory usage
