API Configuration
API token with Zone:Read and DNS:Edit permissions
How to get your API Token
  • Go to Cloudflare Dashboard → API Tokens
  • Click "Create Token"
  • Use the "Edit zone DNS" template or create custom token
  • Ensure Zone:Read and DNS:Edit permissions are granted
  • Select the specific zones you want to manage

Host Configuration
Cloud server Tailscale IP address
Local server Tailscale IP address
Setting up Tailscale
  • Install Tailscale on all servers: curl -fsSL https://tailscale.com/install.sh | sh
  • Authenticate: sudo tailscale up
  • Get IP address: tailscale ip -4
  • View all devices at Tailscale Admin Console
  • Set environment variables: TAILSCALE_LINODE_HOST and TAILSCALE_LOCAL_HOST

SSH Key Configuration
Path to your SSH private key for fleet access
SSH Key Setup
  • Generate SSH key: ssh-keygen -t ed25519 -C "homelab-fleet"
  • Copy public key to servers: ssh-copy-id user@server-ip
  • Set key path: FLEET_SSH_KEY_PATH=/path/to/private/key
  • Set SSH users: FLEET_LINODE_SSH_USER=root, FLEET_LOCAL_SSH_USER=evin
  • Ensure key has proper permissions: chmod 600 ~/.ssh/id_rsa

Storage Configuration
S3-compatible endpoint URL
Getting Storage Credentials
  • Backblaze B2: Create application key at B2 Cloud Storage → App Keys
  • MinIO: Use the access/secret keys from your MinIO deployment
  • AWS S3: Create IAM user with S3 access at AWS IAM Console
  • Set environment variables: MINIO_ENDPOINT, MINIO_ACCESS_KEY, MINIO_SECRET_KEY

API Configuration
Your OpenAI API key (starts with sk-)
Using Replit AI Integration (AI_INTEGRATIONS_OPENAI_API_KEY)
Getting your API Key
  • Go to OpenAI Platform → API Keys
  • Click "Create new secret key"
  • Copy the key immediately (it won't be shown again)
  • Set environment variable: OPENAI_API_KEY=sk-...
  • Or use Replit's built-in AI integration for automatic key management
